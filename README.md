# 💰 Expense Tracker

A full-stack web application built with **Python Flask** and **SQLite** that helps users track their daily expenses efficiently.

## 📌 Features

- ✅ Add new expenses with category and date
- ✅ Edit existing expenses
- ✅ Delete expenses
- ✅ Search expenses by name or category
- ✅ View monthly summary
- ✅ Dashboard with total expense calculation
- ✅ Responsive and professional UI
- ✅ SQLite database for data persistence

## 🚀 How to Run This Project Locally

### Step 1: Clone the repository
git clone https://github.com/Mannatbatish/expense-tracker.git

### Step 2: Go into the project folder
cd expense-tracker

### Step 3: Install dependencies
pip install flask

### Step 4: Run the application
python app.py

### Step 5: Open in browser
http://127.0.0.1:5000
